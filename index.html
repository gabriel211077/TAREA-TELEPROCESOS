Introducción al Teleproceso

Tarea Escolar: Comprendiendo la comunicación de datos a distancia

¿Qué es el Teleproceso?

El término proviene de la unión de Telecomunicación y Procesamiento de Datos. Se define como la técnica que permite la ejecución de procesos de cómputo en una ubicación remota respecto a donde se generan los datos.


En este sistema, una computadora central (Host) sirve recursos a diversos terminales distribuidos geográficamente, utilizando redes de comunicación públicas o privadas.

Arquitectura del Sistema

Todo sistema de teleproceso requiere de tres elementos fundamentales para establecer el flujo de información:

DTE
(Computadora)
DCE
(Módem)
CANAL
(Red)
Componente Función Técnica
DTE (Data Terminal Equipment) Equipo que genera o recibe los datos finales.
DCE (Data Circuit Equipment) Dispositivo que adapta la señal al medio (Módem).
CANAL El medio físico (Fibra, Satélite, Radio).

Modos de Transmisión

La dirección y simultaneidad del flujo de datos se clasifica en tres tipos:

Simplex

La comunicación es unidireccional. Un equipo solo envía y el otro solo recibe (Ej: Radio).

Half-Duplex

Bidireccional, pero no simultánea. Se debe esperar a que el canal esté libre (Ej: Walkie-talkie).

Full-Duplex

Comunicación bidireccional simultánea. Es el estándar de la web moderna.
